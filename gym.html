☁️ Cloud Sync
Sync your data across all devices using Supabase (free).

1. Go to supabase.com → create a free account & new project
2. In your project → SQL Editor → paste & run this SQL:

create table workouts ( id text primary key, user_id uuid references auth.users not null, session_id text, date text, exercise text, cat text, muscles jsonb, sets jsonb, created_at timestamptz default now() ); create table body_weights ( user_id uuid references auth.users not null, date text not null, weight float not null, primary key (user_id, date) ); create table custom_exercises ( id uuid primary key default gen_random_uuid(), user_id uuid references auth.users not null, name text, cat text, muscles jsonb ); alter table workouts enable row level security; alter table body_weights enable row level security; alter table custom_exercises enable row level security; create policy "own" on workouts for all using (auth.uid()=user_id); create policy "own" on body_weights for all using (auth.uid()=user_id); create policy "own" on custom_exercises for all using (auth.uid()=user_id);

3. Settings → API → copy Project URL and anon/public key
4. Click Sign Up tab above and paste them in
